Powerful tools

Rethink how you build games

Godot provides a huge set of common tools, so you can focus on making your game without reinventing the wheel. Godot’s innovative Node and Scene system give you both power and flexibility to create anything.

Dedicated 2D engine

Build crisp and performant 2D games with Godot's dedicated 2D rendering engine with real 2D pixel coordinates and 2D nodes. Don't settle for bolted on 2D.

Learn more

Simple and powerful 3D

Godot's 3D nodes give you everything you need to build, animate, and render your 3D worlds and characters.

Polyglot tool

Use the right language for the job

Keep your code modular with an Object oriented API using Godot's own GDScript, C#, C++, or bring your own using GDNative. With Cross-language scripting, you can get the best of all worlds by writing expressive code in GDScript, and performance critical pieces in C# or C++.

Learn more

Godot is 100% free and fully Open Source

Godot is completely free and open-source under the permissive MIT license. Your game is yours, down to the last line of engine code. No royalties, no strings attached.
